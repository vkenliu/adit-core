# ADIT Label

Manage labels on timeline events:

- Add a label: `adit label <id> <label>`
- Remove a label: `adit label remove <id> <label>`
- List all labels: `adit label list`
- List events with a label: `adit label list --label <name>`

Labels must be alphanumeric with hyphens, max 50 characters.
